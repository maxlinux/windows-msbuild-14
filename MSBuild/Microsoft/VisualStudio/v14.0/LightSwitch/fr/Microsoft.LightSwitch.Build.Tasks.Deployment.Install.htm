Instructions d'installation de l'application Microsoft Visual Studio LightSwitch

  Les étapes suivantes sont nécessaires pour installer une application de bureau cliente à 2 niveaux :

1.    Installez SQL Server Express sur l'ordinateur cible (client), s'il n'est pas déjà installé. Les applications LightSwitch nécessitent soit SQL Server 2005 ou version ultérieure, soit SQL Server Express 2005 ou version ultérieure. En savoir plus sur les options d'installation de SQL Server Express.

2.    Sur l'ordinateur cible, déployez le schéma de la base de données à l'aide de l'outil en ligne de commande sqlcmd.exe. L'outil Sqlcmd.exe est installé avec SQL Server, généralement dans le répertoire \Microsoft SQL Server\110\Tools\Binn du répertoire Program Files ou Program File (x86).

a.     À partir d'une invite de commandes avec élévation de privilèges, accédez au répertoire contenant la sortie de publication, et entrez une commande similaire à ce qui suit :

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Remplacez MyApplication par le nom de votre application et remplacez .\SQLExpress par l'emplacement de la base de données sur le serveur.

3.    Localisez le fichier web.config dans le sous-répertoire Fichiers d'application de la sortie de publication.

a.     Ouvrez le fichier web.config dans le Bloc-notes et modifiez la connexion dans la section <connectionStrings> de manière à utiliser le nom de votre serveur et de votre base de données. Par exemple, pour une entrée similaire à :

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Remplacez SERVERNAME par le nom du serveur de base de données, et MyApplication par le nom de votre application.

4 .    Exécutez Setup.exe sur l'ordinateur cible pour installer l'application.
