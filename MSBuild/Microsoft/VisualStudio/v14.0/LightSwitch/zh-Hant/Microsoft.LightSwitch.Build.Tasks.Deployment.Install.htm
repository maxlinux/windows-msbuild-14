Microsoft Visual Studio LightSwitch 應用程式安裝指令

  以下是安裝兩層桌面用戶端應用程式 的必要步驟:

1.    在目標 (用戶端) 電腦上安裝 SQL Server Express (若無安裝)， LightSwitch 應用程式需要 SQL Server 2005 或更新版本， 或 SQL Server Express 2005 或更新版本。 進一步了解 SQL Server Express 安裝選項。

2.    在目標電腦上使用 sqlcmd.exe 命令工具部署 資料庫結構描述。Sqlcmd.exe 工具會和 SQL Server 一起安裝， 且通常位於 Program Files 或 Program File (x86) 目錄下的 \ Microsoft SQL Server\110\Tools\Binn 目錄中。

a.     從更高權限的命令提示字元，變更為 包含發行輸出的目錄，並輸入類似以下的 命令:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 以您的應用程式名稱來取代 MyApplication，並以伺服器上的資料庫位置取代 .\SQLExpress。

3.    在發行輸出的 Application Files 子目錄中 找出 web.config 檔案。

a.     使用「記事本」開啟 web.config，並編輯 <connectionStrings> 中的連接。 使用您伺服器和資料庫名稱的區段。 例如，類似以下的輸入:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

將 SERVERNAME 替換為資料庫伺服器名稱，並將 MyApplication 替換為您的應用程式名稱。

4 .    在目標電腦上執行 Setup.exe， 以安裝應用程式。
