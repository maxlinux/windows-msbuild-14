Pokyny pro instalaci prostředí Microsoft Visual Studio LightSwitch

  K nainstalování dvouvrstvé desktopové klientské aplikace je potřeba udělat tyto kroky:

1.    Nainstalujte na cílovém (klientském) počítači SQL Server Express, pokud ještě není nainstalovaný. Aplikace LightSwitch vyžadují SQL Server 2005 nebo novější verzi, nebo SQL Server Express 2005 nebo novější verzi. Přečtěte si další informace o možnostech instalace SQL Serveru Express.

2.    Pomocí nástroje příkazového řádku sqlcmd.exe nasaďte na cílovém počítači schéma databáze. Nástroj sqlcmd.exe se instaluje s SQL Serverem, obvykle do adresáře \Microsoft SQL Server\110\Tools\Binn pod adresářem Program Files nebo Program Files (x86).

a.     Na příkazovém řádku se zvýšenými oprávněními změňte adresář na ten, který obsahuje výstup publikování, a zadejte příkaz podobný tomuto:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Nahraďte MojeAplikace názvem vaší aplikace, poté nahraďte .\SQLExpress umístěním databáze na serveru.

3.    V podadresáři Soubory aplikace výstupu publikování najděte soubor web.config.

a.     Otevřete web.config v Poznámkovém bloku a upravte připojení v oddílu <connectionStrings> tak, aby obsahovalo název vašeho serveru a databáze. Třeba u položky tohoto typu:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

nahraďte SERVERNAME názvem databázového serveru a MyApplication názvem vaší aplikace.

4.     Spusťte soubor Setup.exe na cílovém počítači a nainstalujte aplikaci.
