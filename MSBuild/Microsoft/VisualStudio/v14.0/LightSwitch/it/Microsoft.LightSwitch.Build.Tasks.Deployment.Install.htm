Istruzioni per l'installazione dell'applicazione Microsoft Visual Studio LightSwitch

  I seguenti passaggi sono necessari per installare un'applicazione client desktop a 2 livelli:

1.    Installare SQL Server Express nel computer di destinazione (client), se non è già installato. Le applicazioni LightSwitch richiedono SQL Server 2005 o versione successiva oppure SQL Server Express 2005 o versione successiva. Altre informazioni sulle opzioni di installazione di SQL Server Express.

2.    Nel computer di destinazione distribuire lo schema del database usando lo strumento di comando sqlcmd.exe. Lo strumento sqlcmd.exe viene installato con SQL Server, generalmente nella directory \Microsoft SQL Server\110\Tools\Binn della directory Programmi o Programmi (x86).

a.     Da un prompt dei comandi con privilegi elevati, passare alla directory contenente l'output di pubblicazione e immettere un comando simile al seguente:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Sostituire MyApplication con il nome dell'applicazione e sostituire .\SQLExpress con il percorso del database nel server.

3.    Individuare il file web.config nella sottodirectory Application Files dell'output di pubblicazione.

a.     Aprire web.config nel Blocco note e modificare la connessione nella sezione <connectionStrings> per usare il nome del server e del database desiderati. Ad esempio, per una voce simile a:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Sostituire SERVERNAME con il nome del server di database e sostituire MyApplication con il nome dell'applicazione.

4 .    Eseguire Setup.exe nel computer di destinazione per installare l'applicazione.
