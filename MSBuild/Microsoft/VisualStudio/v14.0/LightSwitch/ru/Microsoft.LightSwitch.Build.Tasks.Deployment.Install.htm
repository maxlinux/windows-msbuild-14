Инструкции по установке приложения Microsoft Visual Studio LightSwitch

  Чтобы установить 2-уровневое классическое клиентское приложение , необходимо выполнить следующие действия:

1.    Установите SQL Server Express на целевой (клиентский) компьютер, если он еще не установлен. Приложения LightSwitch требуют SQL Server 2005 или более позднюю версию, или SQL Server Express 2005 или более позднюю версию. Подробнее о параметрах установки SQL Server Express.

2.    На целевом компьютере разверните схему базы данных с помощью средства командной строки sqlcmd.exe. Средство командной строки Sqlcmd.exe устанавливается вместе с SQL Server, обыкновенно в папке \Microsoft SQL Server\110\Tools\Binn, расположенной в папке Program Files или Program File (x86).

a.     В командной строке с повышенными привилегиями выберите каталог, который содержит выходные данные публикации, и введите аналогичную следующей команду:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Вместо MyApplication подставьте имя приложения, а вместо .\SQLExpress - расположение базы данных на сервере.

3.    Найдите файл web.config в подкаталоге Application Files вывода публикации.

a.     Откройте файл web.config в Блокноте и измените подключение в разделе < connectionStrings>, указав правильные имя сервера и базы данных. Например, для записи следующего вида:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Замените SERVERNAME на имя сервера базы данных, а MyApplication - на имя вашего приложения.

4 .    Запустите программу Setup.exe на целевом компьютере, чтобы установить приложение.
