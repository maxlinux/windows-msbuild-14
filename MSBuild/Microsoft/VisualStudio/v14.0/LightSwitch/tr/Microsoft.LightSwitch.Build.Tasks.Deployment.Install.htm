Microsoft Visual Studio LightSwitch Uygulaması Yükleme Yönergeleri

  Aşağıdaki adımlar 2 katmanlı masaüstü istemci uygulamasını yüklemek için gereklidir:

1.    SQL Server Express'i hedef (istemci) bilgisayarına yükleyin, zaten yüklüyse yüklemeniz gerekmez. LightSwitch Uygulamaları, SQL Server 2005 veya daha sonraki bir sürümü veya SQL Server Express 2005 ya da daha sonraki bir sürümü gerektirir. SQL Server Express yükleme seçenekleri hakkında daha fazla bilgi edinin.

2.    Hedef bilgisayarda sqlcmd.exe komut aracını kullanarak veritabanı şemasını dağıtın. Sqlcmd.exe aracı SQL Server ile genellikle Program Files veya Program File (x86) dizininin altında \Microsoft SQL Server\110\Tools\Binn dizinine yüklenir.

a.     Yükseltilmiş bir Komut İstemi'nden, yayımlama çıktısını içeren dizine geçin ve şuna benzer bir komut girin:

 sqlcmd.exe -i Uygulamam.sql -S .\SQLExpress

 BenimUygulamam yerine uygulamanızın adını yazın ve .\SQLExpress yerine sunucudaki veritabanı konumunu yazın.

3.    Yayım çıktısının Application Files alt dizininde web.config dosyasını bulun.

a.     web.config dosyasını Not Defteri'nde açın ve sunucu ve veritabanı adını kullanmak için <connectionStrings> dizesindeki bağlantıyı düzenleyin. Örneğin şuna benzer bir girdi:

<add name="_IntrinsicData" connectionString="Data Source=SUNUCUADI\SQLEXPRESS;Initial Catalog=Uygulamam;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

SUNUCUADI 'nı veritabanı sunucusunun adıyla ve Uygulamam'ı ise uygulamanızın adıyla değiştirin.

4 .    Uygulamayı yüklemek için hedef bilgisayarda Setup.exe'yi çalıştırın.
