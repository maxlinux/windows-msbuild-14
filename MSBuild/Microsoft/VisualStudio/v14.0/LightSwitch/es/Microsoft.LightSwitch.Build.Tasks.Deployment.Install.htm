Instrucciones de instalación de la aplicación Microsoft Visual Studio LightSwitch

  Se necesitan los pasos siguientes para instalar una aplicación de cliente para escritorio de dos niveles:

1.    Instale SQL Server Express en el equipo de destino (de cliente), si todavía no está instalado. LightSwitch Applications necesita SQL Server 2005 o una versión posterior, o bien SQL Server Express 2005 o una versión posterior. Más información sobre las opciones de instalación de SQL Server Express.

2.    En el equipo de destino, implemente el esquema de base de datos mediante la herramienta de comandos sqlcmd.exe. La herramienta Sqlcmd.exe se instala junto con SQL Server, normalmente en el directorio \Microsoft SQL Server\110\Tools\Binn, en el directorio Archivos de programa o Archivo de programa (x86).

a.     Desde un símbolo del sistema con privilegios elevados, cambie al directorio que contenga la salida de publicación e introduzca un comando similar al siguiente:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Reemplace MyApplication por el nombre de su aplicación y .\SQLExpress por la ubicación de la base de datos en el servidor.

3.    Busque el archivo web.config en el directorio Archivos de la aplicación de la salida de publicación.

a.     Abra web.config en el Bloc de notas y edite la conexión en la sección <connectionStrings> para que el nombre de su servidor y su base de datos. Por ejemplo, para una entrada similar a:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Sustituya SERVERNAME por el nombre del servidor de la base de datos, y sustituya MyApplication por el nombre de su aplicación.

4 .    Ejecute Setup.exe en el equipo de destino para instalar la aplicación.
