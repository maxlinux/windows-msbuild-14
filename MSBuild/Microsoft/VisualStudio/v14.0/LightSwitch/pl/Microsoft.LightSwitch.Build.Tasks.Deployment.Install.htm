Instrukcja instalacji aplikacji Microsoft Visual Studio LightSwitch

  Aby zainstalować dwuwarstwową aplikację klasyczną klienta, wykonaj te kroki:

1.    Zainstaluj oprogramowanie SQL Server Express na komputerze docelowym (klienta), jeśli nie jest ono już zainstalowane. Aplikacje LightSwitch wymagają oprogramowania SQL Server 2005 lub nowszego albo oprogramowania SQL Server Express 2005 lub nowszego. Dowiedz się więcej o opcjach instalacji oprogramowania SQL Server Express.

2.    Na komputerze docelowym wdróż schemat bazy danych przy użyciu narzędzia polecenia sqlcmd.exe. Narzędzie sqlcmd.exe jest instalowane wraz z oprogramowaniem SQL Server, zwykle w podkatalogu \Microsoft SQL Server\110\Tools\Binn katalogu Program Files lub Program File (x86).

a.nbsp;    W wierszu polecenia z podwyższonym poziomem uprawnień przełącz katalog, który zawiera publikowane dane wyjściowe, i wprowadź polecenie podobne do następującego:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

nbsp;Zastąp moja_aplikacja nazwą swojej aplikacji i zastąp .\SQLExpress lokalizacją bazy danych na serwerze.

3.nbsp;nbsp;  Zlokalizuj plik web.config w podkatalogu Pliki aplikacji publikowanych danych wyjściowych.

a.nbsp;nbsp;   Otwórz plik web.config w Notatniku i zmień połączenie w sekcji <connectionStrings>, aby użyć nazwy serwera i nazwy baza danych. Na przykład, dla wpisu podobnego do:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Zastąp SERVERNAME nazwą serwera bazy danych, a MyApplication nazwą swojej aplikacji.

4 .    Uruchom plik Setup.exe na komputerze docelowym, aby zainstalować aplikację.
