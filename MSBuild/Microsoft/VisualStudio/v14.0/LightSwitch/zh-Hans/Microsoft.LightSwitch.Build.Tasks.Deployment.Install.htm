Microsoft Visual Studio LightSwitch 应用程序安装说明

  以下步骤是安装 2 层桌面 客户端应用程序的必要步骤：

1.    如果尚未安装 SQL Server Express，则将其安装在目标（客户端） 计算机上。LightSwitch 应用程序需要 SQL Server 2005 或更高 版本，或者 SQL Server Express 2005 或更高版本。 了解更多

2.    在目标计算机上，通过使用 sqlcmd.exe 命令工具部署 数据库架构。Sqlcmd.exe 通常安装在 SQL Server 中的 \Microsoft SQL Server\110\Tools\Binn 目录（在 Program Files or Program File (x86) 目录下）。

a.     从提升的命令提示符下，更改为包含发布输出的 目录，然后输入如下命令 ：

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 用您的应用程序名称替换 MyApplication，并用服务器上的数据库位置替换 .\SQLExpress。

3.    在发布输出的应用程序文件 子目录中查找 web.config 文件。

a.     在记事本中打开 web.config，在 <connectionStrings> 中编辑连接 使用您的服务器和数据库名称。例如， 输入如下内容：

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

将 SERVERNAME 替换为数据库服务器的名称，将 MyApplication 替换为您的应用程序名称。

4 .    在目标计算机上运行 Setup.exe 以 安装此应用程序。
