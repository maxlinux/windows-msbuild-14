Installationsanweisungen für die Microsoft Visual Studio LightSwitch-Anwendung

  Die folgenden Schritte sind erforderlich, um eine 2-stufige Clientanwendung für Desktops zu installieren:

1.    Installieren Sie SQL Server Express auf dem Zielcomputer (Client-), sofern dies noch nicht geschehen ist. LightSwitch-Anwendungen erfordern SQL Server 2005 oder eine höhere Version, oder SQL Server Express 2005 oder eine höhere Version. Weitere Informationen zu den Installationsoptionen von SQL Server Express finden Sie hier.

2.    Stellen Sie auf dem Zielcomputer das Datenbankschema bereit, indem Sie das Befehlstool "sqlcmd.exe" verwenden. "sqlcmd.exe" wird mit SQL Server installiert, üblicherweise im Verzeichnis "\Microsoft SQL Server\110\Tools\Binn" unter "Programme" oder "Programme (x86)".

a.     Wechseln Sie von einer Eingabeaufforderung mit erhöhten Rechten in das Verzeichnis, das die Veröffentlichungsausgabe enthält, und geben Sie einen Befehl ein, der dem folgenden ähnelt:

 sqlcmd.exe -i MeineAnwendung.sql -S .\SQLExpress

 Ersetzen Sie MyApplication durch den Namen Ihrer Anwendung, und ersetzen Sie .\SQLExpress durch den Datenbankspeicherort auf dem Server.

3.    Suchen Sie nach der Datei "web.config" im Unterverzeichnis "Application Files" der Veröffentlichungsausgabe.

a.     Öffnen Sie "web.config" in Editor, und bearbeiten Sie die Verbindung im Abschnitt <connectionStrings> so, dass der Name Ihres Servers und der Name Ihrer Datenbank verwendet werden. Nehmen Sie folgenden Eintrag als Beispiel:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MeineAnwendung;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Ersetzen Sie SERVERNAME durch den Namen des Datenbankservers, und ersetzen Sie MeineAnwendung durch den Namen Ihrer Anwendung.

4 .    Führen Sie "Setup.exe" auf dem Zielcomputer aus, um die Anwendung zu installieren.
