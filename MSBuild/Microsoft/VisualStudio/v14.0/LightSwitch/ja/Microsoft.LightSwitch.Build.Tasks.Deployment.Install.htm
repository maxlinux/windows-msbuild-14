Microsoft Visual Studio LightSwitch アプリケーションのインストール手順

  次の手順は、2 層のデスクトップ クライアント アプリケーションの インストールに必要です

1.    SQL Server Express がまだインストールされていない場合は、対象の (クライアント) コンピューター にインストールします。LightSwitch アプリケーションには、SQL Server 2005 以上のバージョン、 または SQL Server Express 2005 以上のバージョンが必要です。 SQL Server Express のインストール オプションについて詳細を確認してください。

2.    対象のコンピューターで、sqlcmd.exe コマンド ツールを使用してデータベースのスキーマを 配置します。sqlcmd.exe ツールは、一般に Program Files または Program File (x86) ディレクトリの下の \Microsoft SQL Server\110\Tools\Binn ディレクトリに SQL Server と一緒にインストールされます。

a.     管理者特権でのコマンド プロンプトで、発行の出力を含む ディレクトリに変更し、次のようなコマンドを入力します。

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 MyApplication はアプリケーションの名前に置き換え、 .\SQLExpress はサーバー上のデータベースの場所に置き換えます。

3.    発行の出力の Application Files サブディレクトリで web.config ファイルを探します。

a.     web.config をメモ帳で開き、サーバーおよびデータベース名 を使用するために、<connectionStrings> セクションの接続を編集します。たとえば 次のような入力内容になります。

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

SERVERNAME をデータベース サーバーの名前に置き換え、 MyApplication をアプリケーション名に置き換えます。

4 .    対象のコンピューターで Setup.exe を 実行して、アプリケーションをインストールします。
