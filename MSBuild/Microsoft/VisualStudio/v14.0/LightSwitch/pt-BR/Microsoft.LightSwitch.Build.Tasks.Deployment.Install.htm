Instruções de instalação da aplicação Microsoft Visual Studio LightSwitch

  Os seguintes passos são necessários para instalar um aplicativo cliente de área de trabalho com duas camadas:

1.    Instale o SQL Server Express no computador de destino (cliente), se ainda não estiver instalado. LightSwitch Applications requer o SQL Server 2005 ou uma versão mais recente, ou o SQL Server Express 2005 ou uma versão mais recente. Saiba mais sobre as opções de instalação do SQL Server Express.

2.    No computador de destino, implante o esquema de banco de dados usando a ferramenta de comando sqlcmd.exe. A ferramenta Sqlcmd.exe é instalada com o SQL Server, normalmente no diretório \Microsoft SQL Server\110\Tools\Binn no diretório Program Files ou Program File (x86).

a.     A partir de um prompt de comando elevado, mude para o diretório que contém a saída para publicar e inserir um comando semelhante ao seguinte:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Substitua MyApplication pelo nome de sua aplicação e substitua .\SQLExpress pela localização do banco de dados no servidor.

3.    Localize o arquivo web.config no subdiretório Application Files da saída de publicação.

a.     Abra web.config no Notepad e edite a conexão em <connectionStrings> seção para usar seu servidor e o nome do banco de dados. Por exemplo, para a entrada semelhante a:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Substitua SERVERNAME pelo nome do servidor de banco de dados e substitua MyApplication pelo nome de sua aplicação.

4 .    Execute Setup.exe no computador de destino para instalar a aplicação.
